# Yost Group LLC — Website

Public website for [Yost Group LLC](https://www.yost.group), a Service Disabled Veteran Owned Small Business (SDVOSB) providing program management consulting and financial coaching for couples.

## Structure

```
/
├── index.html          # Home
├── consult.html        # Program management consulting
├── coach.html          # Financial coaching for couples
├── about.html          # About Rick Yost
├── blog.html           # Blog index
├── course.html         # Course page
├── monthly.html        # Monthly coaching package
├── package.html        # Coaching package detail
├── articles/           # Blog posts
│   ├── blog-money-talk.html
│   ├── blog-coach-vs-advisor.html
│   ├── blog-high-income-broke.html
│   └── blog-couples-budget.html
├── resources/          # Images and media
│   ├── Rick_Yost3.jpg
│   ├── rick-photo.jpg
│   └── sba-badge.jpg
├── sitemap.xml
├── robots.txt
└── CNAME
```

## Tech

Single-file HTML pages with embedded CSS. No build process, no frameworks, no external dependencies beyond Google Fonts CDN.

- Google Analytics 4 (`G-SYH15WF50B`)
- Schema.org structured data (Organization, Person, LocalBusiness, BlogPosting)
- Canonical URLs and Open Graph tags on all pages

## Deployment

Hosted via GitHub Pages. The `CNAME` file points to `www.yost.group`. Pushing to `main` deploys automatically.
